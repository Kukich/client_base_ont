Настройки БД и хоста находятся в файле MyConf.pm.
Запросы на создание БД и таблицы clients находятся в sql файле
Для запуска приложения надо запустить server.pl и открыть в браузере http://localhost:8080/main.cgi (если использовать настройки по умолчанию)
Для запуска тестов(test_HTMLCode.pl и test_DateCalc) необходимы модули Test::More и Test::Mockmodule
